From Learning to Prediction: House Prices Regression

An end-to-end machine learning project to predict house sale prices — putting newly acquired data science skills to the test on the Ames, Iowa housing dataset.

Type
Personal Project
Focus
Machine Learning
Stack
Python · Pandas · Scikit-learn
Dataset
Ames, Iowa Housing

The starting point

My data science journey began with a comprehensive course that built a strong foundation in the essential tools of the trade — manipulating data with Pandas, numerical operations with NumPy, visualisation with Matplotlib and Seaborn, and applying ML algorithms with Scikit-learn.

Eager to apply these skills to a real problem, I turned to Kaggle and the Ames Housing dataset — the foundation for this project.

The goal: predicting house prices

The objective was to build a regression model that could accurately predict the sale price of a house from its features. With 79 variables — from lot area and bedrooms to construction quality — it was an ideal challenge requiring a systematic approach to cleaning, analysis and modelling.

3 · Model building & evaluation

I built and compared several regression models, using a helper to streamline training and evaluation.

def run_model(model):
    model.fit(X_train, y_train)
    y_preds = model.predict(X_test)
    MAE = mean_absolute_error(y_test, y_preds)
    RMSE = np.sqrt(mean_squared_error(y_test, y_preds))
    print(f'MAE: {MAE}, RMSE: {RMSE}')

Gradient Boosting consistently performed best. After tuning, it achieved an RMSE around 12.5% of the mean sale price — a strong, accurate result.

run_model(GradientBoostingRegressor(n_estimators=1000, max_depth=2))

Beyond the project: real-world applications

More than a learning exercise, this project showed how regression can solve real business problems:

  • Real estate valuation — pricing properties competitively and identifying undervalued assets.
  • Financial services — automating property appraisals to speed up mortgage applications.
  • Other industries — retail sales forecasting, insurance claim-cost prediction, and manufacturing materials pricing all use similar techniques.

Ultimately, it solidified my understanding of the end-to-end data science process and the potential of ML to drive informed decisions.
