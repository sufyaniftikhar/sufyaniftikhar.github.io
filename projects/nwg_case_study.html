Project Overview

Northumbrian Water Group (NWG) required enhanced visibility into their contract performance across their AMP7 (2020-2025) and AMP8 (2025-2030) Asset Management Periods. While their existing Contract Bee system captured comprehensive contract data, it lacked the analytical depth and custom visualisation capabilities needed for strategic decision-making.

As the lead data analyst, I designed and deployed a comprehensive Power BI solution that transformed how NWG's commercial teams monitor contract performance, manage compensation events, and ensure regulatory compliance across their £3 billion infrastructure programme.

Business Challenge: Contract Bee's standard reporting couldn't provide the custom analytics, interactive filtering, and real-time insights needed by commercial managers, project teams, and compliance officers to effectively oversee 300+ active contracts.

Technical Architecture

The solution leveraged Contract Bee's API for real-time data integration, eliminating manual export processes and ensuring stakeholders always accessed current contract information.

Power BI Contract Bee API Power Query DAX Power Automate Power Platform

Technical Challenges Addressed

  • API rate limiting and connection optimization for real-time data refresh
  • Complex data relationships across multiple contract lifecycle stages
  • Performance optimization for large datasets (300+ contracts with historical data)
  • Custom DAX calculations for contract-specific KPIs and variance analysis
  • Multi-user access controls and workspace security implementation

Development Methodology

Following agile principles, the project was delivered through a structured five-phase approach, ensuring stakeholder engagement and iterative refinement throughout the development lifecycle.

1

Requirements Gathering

Conducted stakeholder workshops with Senior Commercial Managers and Project Managers to define reporting requirements, identify key metrics, and establish success criteria for the dashboard solution.

2

Data Integration & Modeling

Established Contract Bee API connectivity, designed optimal data models with proper relationships, and created custom DAX measures for contract performance calculations and variance analysis.

3

Dashboard Development

Built interactive visualizations with advanced filtering capabilities, drill-down analysis features, and responsive design optimized for both desktop and mobile access across different user roles.

4

User Testing & Refinement

Conducted iterative testing sessions with end-users, gathered feedback on usability and performance, and implemented optimizations to enhance user experience and system responsiveness.

5

Deployment & Training

Deployed the solution to NWG's Commercial Function workspace, implemented security controls, and delivered comprehensive training programs to ensure successful adoption across all user groups.

Key Features Delivered

The final solution provided comprehensive contract oversight capabilities tailored to NWG's operational requirements:

Contract Performance Tracking: Real-time monitoring of contract status, financial commitments, and milestone progress with automated alerts for overdue actions and compensation events.
Financial Analysis: Dynamic variance analysis comparing contracted values against actual spend, with detailed breakdown of compensation events and their financial impact on programme budgets.
Risk Management: Automated identification of contracts with overdue early warnings, pending compensation events, and compliance issues requiring immediate management attention.
Interactive Analytics: Advanced filtering by contractor, project manager, contract type, and timeline with drill-through capabilities for detailed contract investigation and root cause analysis.
